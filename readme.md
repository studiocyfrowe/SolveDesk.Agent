## SOLVEDESK AGENT

# REST API
Narzędzie diagnostyczne

Przykłady użycia:
1) Komputer działa bardzo wolno → SPRAWDŹ WYDAJNOŚĆ KOMPUTERA (CPU, RAM, dysk)
2) Nie można zalogować się do aplikacji → SPRAWDŹ, CZY AKTYWNA JEST USŁUGA APLIKACJI
3) Brak połączenia z internetem → SPRAWDŹ STATUS KARTY SIECIOWEJ ORAZ KONFIGURACJĘ IP
4) Aplikacja nie uruchamia się → SPRAWDŹ LOGI APLIKACJI ORAZ STATUS PROCESU
5) Brak dostępu do dysku sieciowego → SPRAWDŹ POŁĄCZENIE Z SERWEREM ORAZ UPRAWNIENIA UŻYTKOWNIKA
6) Drukarka nie drukuje → SPRAWDŹ STATUS USŁUGI DRUKOWANIA ORAZ KOLEJKĘ WYDRUKU
7) Użytkownik nie ma dostępu do folderu → SPRAWDŹ UPRAWNIENIA NTFS ORAZ PRZYNALEŻNOŚĆ DO GRUP
8) Aplikacja działa bardzo wolno → SPRAWDŹ OBCIĄŻENIE SERWERA ORAZ CZAS ODPOWIEDZI BAZY DANYCH
9) Komputer często się restartuje → SPRAWDŹ LOGI SYSTEMOWE ORAZ BŁĘDY W PODGLĄDZIE ZDARZEŃ
10) Nie działa poczta w Outlooku → SPRAWDŹ POŁĄCZENIE Z SERWEREM POCZTY ORAZ STATUS KONTA
